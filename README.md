## 🚀 Как использовать

### Генерация пояснительной записки

1. Откройте **CHATGPT_PROMPT.md**
2. Скопируйте главный промпт
3. Вставьте в ChatGPT или Claude
4. Получите полный текст дипломной работы

### Генерация диаграмм

**Вариант 1: Использовать готовые**
1. Откройте **diagrams.html** в браузере
2. Сделайте скриншоты диаграмм
3. Вставьте в документ Word

**Вариант 2: Сгенерировать через ChatGPT**
1. Откройте **DIAGRAM_PROMPTS.md**
2. Скопируйте промпт нужной диаграммы
3. Вставьте в ChatGPT
4. Получите код Mermaid
5. Визуализируйте на https://mermaid.live/

**Вариант 3: Использовать PlantUML**
1. Откройте **DIAGRAMS.md**
2. Скопируйте PlantUML код диаграммы
3. Вставьте на https://www.plantuml.com/plantuml/uml/
4. Экспортируйте в PNG/SVG

## 📋 Структура системы

### Роли пользователей (4 роли)

1. **Гость** — просмотр каталога, поиск, фильтрация, сравнение, регистрация
2. **Пользователь** — создание заявок, просмотр своих заявок, отмена, оплата
3. **Администратор** — управление услугами, предприятиями, обработка заявок
4. **Менеджер по финансам** — аналитика, финансы, CRM (клиенты, счета, скидки, бюджеты)

### Технологии

- **Backend:** Python 3.8+, Flask 2.3+, pyodbc
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, Chart.js
- **База данных:** MS SQL Server 2016+ (11 таблиц)
- **Архитектура:** Трёхуровневая клиент-серверная

### База данных (11 таблиц)

1. Users — пользователи
2. ServiceCategories — категории услуг
3. Plants — предприятия
4. Services — услуги
5. Requests — заявки
6. RequestStatusHistory — история статусов
7. Clients — клиенты (CRM)
8. Invoices — счета
9. PaymentHistory — платежи
10. Discounts — скидки
11. MonthlyBudgets — бюджеты
